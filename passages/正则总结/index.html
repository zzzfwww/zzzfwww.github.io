三剑客和正则总结

发布 : 2022-01-15 浏览 :

正则注意事项

  • 所有的符号英文符号
  • 学习正则通过grep命令学习，grep加上单引号
  • 给grep,egrep加上颜色 alias grep=’grep –color=auto’ alias egrep=’egrep –color=auto’
  • 注意系统的字符集：en_US.UTF-8(大部分情况没问题)，如果出现问题修改字符集为C export LANG=C
  • 快速掌握正则：配合grep -o 参数学习

sed命令结合正则

1） sed-查找p

查找格式 描述
‘2p’ 指定行号进行查找
’1,5p’ 指定行号范围进行查找
‘/lidao/p’ 类似于grep，过滤，//里面可以写正则
‘/10:00/,/11:00/p’ 表示范围的过滤
1,/old/ 混合

2） sed 删除

 - d delete
查找格式 描述
‘2p’ 指定行号进行删除
’1,5d’ 指定行号范围进行删除
‘/lidao/d’ 类似于grep，过滤，//里面可以写正则
‘/10:00/,/11:00/d’ 表示范围的过滤
1,/old/ 混合

  • 删除文件中的空行和包含#号的行

3） sed -增加cai

命令 描述
c replace替代这行的内容
a append 追加，向指定的行或每一行追加内容（行后面）>>
i insert 插入，向指定的行或每一行插入内容（行前面）

  • 向文件中追加多行内容

4） sed -替换s

- s->sub substitute替换
- g ->global 全局替换，sed替换每行所有匹配的
替换格式 描述
s###g 替换格式推荐
s@@@g 推荐
s///g 不推荐

  • 反向引用，后向引用

  • 口诀：先保护，再使用

  • -w [0-9a-zA-Z_]

  • sed
